* /images/ - фон, иконка;
* /styles/style.css - файл стилей;
* connect_db.php - установка связи с бд;
* index.php - страничка с формой авторизации (скрывается после успешно введенных данных) и меню пользователя (скрывается после успешно введенных данных);
* auth.js - содержит ф-ции "спрятать/показать форму авторизации", "спрятать/показать меню пользователя",
авторизация" (кидает данные для обработки в auth.php), "завершение сессии" (кидает данные для обработки в quit.php);
* auth.php - принимает данные, проверяет заполененость полей, конектится с базой, сверяет полученные данные с имеющимися, пишет результат;
* quit.php - принимает данные, завершает сессию;

* test_login.sql - на всякий случай
